1 Introduction

Selectomatic is sql which has an sqlite database as input that writes html checkboxes enabling selection of fields in tables and views. Selecting checkboxes then generates an sql select string which can be sent off to get the results

2 Installation

3 Synopsis

ideally

sqlite3 db < selectomatic.sql > htmlFragment

but perhaps

selectomatic -d db > htmlFragment

where htmlFragment looks something like

<fieldset>
<legend>table1</legend>
<label>
    <input type="checkbox" onclick="do_applicableTags(this)" name="table1" value="field1"></input>
    field1
</label>
<label>
    <input type="checkbox" onclick="do_applicableTags(this)" name="table1" value="field2"></input>
    field2
</label>
</fieldset>

<fieldset>
<legend>view1</legend>
<label>
    <input type="checkbox" onclick="do_applicableTags(this)" name="view1" value="field1"></input>
    field1
</label>
<label>
    <input type="checkbox" onclick="do_applicableTags(this)" name="view1" value="field2"></input>
    field2
</label>
</fieldset>
<!--

<textarea id="sqlText" placeholder="sql goes here"></textarea>
